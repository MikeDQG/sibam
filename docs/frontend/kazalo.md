# Kazalo frontend dokumentacije

Ta dokument pove, v kateri frontend dokumentaciji lahko najdemo posamezne vsebine.

## Dokumenti

- `poti/specifikacija-poti.md` opisuje prikaz poti na zemljevidu, polyline odseke, ikone na poti, popupe, step-e, aktivni step, sledenje poti, podatkovni model poti in tok iskanja poti.
- `zemljevid/zemljevid.md` opisuje prikaz glavnega zemljevida, uporabljeno Google Maps knjiznico, parametre komponente `MainMap`, nastavitve `Map`, uporabo `AdvancedMarker`, `InfoWindow` in risanje poti.
- `shranjevanje-poti-in-lokacij/shranjevanje-poti-in-lokacij.md` opisuje shranjevanje lokacij, shranjevanje poti, hitro brisanje shranjene lokacije na zemljevidu in brisanje shranjenih podatkov prek API klicev.
- `mockanje-lokacije-uporabnika/mock-lokacija.md` opisuje, kako lahko za lokalno testiranje rocno mockamo trenutno lokacijo uporabnika v `MainAppHome.applyUserLocation`.
